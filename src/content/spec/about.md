# 关于本站

本站使用 **Astro** 框架并基于 [Mizuki](https://github.com/matsuzaka-yuki/mizuki) 主题构建。

::github{repo="zqwerrg/blog"}

## 🌟 主题功能

### 🎨 设计与用户体验
- **现代且优雅的设计** - 干净、极简的界面以及优美的排版
- **完全响应式** - 针对从移动设备到桌面的所有设备进行优化
- **暗/亮模式** - 自动主题切换并带有平滑过渡
- **精美的排版** - 使用 JetBrains Mono 字体提升可读性
- **流畅的动画** - 平滑的页面过渡和交互元素

### 🔍 内容与搜索
- **高级搜索** - 由 [Pagefind](https://pagefind.app/) 提供支持，搜索快速且准确
- **增强的 Markdown** - 扩展语法，支持代码高亮与数学公式
- **交互式目录** - 为长篇文章提供自动滚动导航
- **RSS 订阅生成** - 自动生成订阅源以便订阅更新
- **阅读时长估算** - 让读者知道文章的大致阅读时长
- **文章分类** - 使用标签和分类来组织内容

### 🌐 国际化
- **多语言支持** - 实时翻译功能
- **自动语言检测** - 自动检测用户偏好语言
- **客户端翻译** - 使用 Edge Translate 技术进行客户端翻译
- **支持 10+ 语言** - 包括 EN、ZH-CN、ZH-TW、JA、KO、ES、TH、VI、ID、TR 等

### 📱 特殊页面
- **番剧跟踪页面** - 跟踪您的番剧观看进度并带评分功能
- **友链页面** - 使用精美卡片展示朋友网站
- **日记/动态页面** - 像社交媒体一样分享生活片段
- **归档页面** - 按时间线组织的所有文章视图
- **关于页面** - 可自定义的个人介绍（就是此页面！）

### 🛠 技术特性
- **增强的代码块** - 使用 [Expressive Code](https://expressive-code.com/) 提升代码展示效果
- **数学公式支持** - 使用 KaTeX 渲染 LaTeX 数学表达式
- **图片优化** - PhotoSwipe 画廊与懒加载支持
- **SEO 优化** - 内置站点地图和 meta 标签以改善搜索能见度
- **性能优化** - 通过缓存与优化实现快速加载
- **评论系统就绪** - 支持与 Twikoo 等评论系统集成

### 🎯 高级 Markdown 功能
- **提示框与注意事项** - 可以写成 `> [!NOTE]`、`> [!TIP]`、`> [!WARNING]` 等漂亮的信息框
- **数学公式** - 使用 `$inline$` 与 `$$block$$` 语法书写 LaTeX 表达式
- **GitHub 卡片** - 使用 `::github{repo="user/repo"}` 嵌入仓库卡片
- **语法高亮** - 高级代码高亮并支持行号
- **复制代码按钮** - 一键复制代码功能
